1. Folder: “Saved_models”
This folder contains the trained CNN model and GBC model.

2. File "Input_data.csv"
This file is the dataset used for training CNN model and GBC model.

3. File "CNN_GBC_model.ipynb"
This file is the original model-training notebooks. They can be opened and executed using Jupyter Notebook.

4. File "CNN_GBC_model.html"
This file provides a complete demonstration of a full model-training process.

5. File "CNN_GBC_model.py"
This file is the script for training the CNN model and GBC model, which is similar to file "CNN_GBC_model.ipynb"
