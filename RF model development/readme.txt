• File “Input_data.csv”
This file contains the dataset used for training the RF model. For each data record, the first five values represent the historical predicted states, while the last two values correspond to the ground-truth state and the state predicted by the CNN-GBC model, respectively.

• File “random_forest_model_best.pkl”
This file stores the trained RF model.

• File “RF_model.html”
This file provides a complete demonstration of the RF model training process.

• File “RF_model.ipynb”
This file is the original Jupyter Notebook used for training the RF model and can be opened and executed directly.

• File "RF_model.py"
This file is the script used for training the RF model.
