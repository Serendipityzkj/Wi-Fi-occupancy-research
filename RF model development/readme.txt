• File “Input_data_RF.csv”
This file contains the dataset used for training the Random Forest (RF) model, including ground-truth occupancy states collected during real operation.

• File “random_forest_model_best.pkl”
This file stores the trained RF model.

• File “RF_model.html”
This file provides a complete demonstration of the RF model training process.

• File “RF_model.ipynb”
This is the original Jupyter Notebook used for training the RF model and can be opened and executed directly.
